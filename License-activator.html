Project Information

  • Project Date: June 2025

IceCode was developing a license activator for their Windows applications. I built both the client-side and server-side components. The client-side is a PowerShell script that collects the device signature and sends a license request to the server. The server, hosted on a Debian Linux machine provided by an external cloud provider, runs Python scripts that allow IceCode admins to approve or deactivate license requests. Once a request is approved, the server handles the license delivery through a confidential method, ensuring secure activation for the user.
